# WAR

browser strategy game. military-themed catan with online multiplayer.

play now: https://war-frontend-production.up.railway.app

## run locally

```bash
npm install
npm run dev
```

for online multiplayer, start the server:

```bash
npm run server
```

## build

```bash
npm run build
```

## deploy

```bash
docker compose up -d
```

## test

```bash
npm test
```

## play

- local or online
- public lobbies or private room codes
- 5 map layouts
- 2-4 players
